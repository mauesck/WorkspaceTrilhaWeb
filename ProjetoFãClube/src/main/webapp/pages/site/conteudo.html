Comandos de Inicialização

Comando Descrição
-novid Retira o vídeo inicial que aparece a logo do Counter-Strike: Global offensive e leva direto para o jogo.
-high O computador define o Counter-Strike como prioridade.
-nod3d9ex1 Otimiza o processo Alt+Tab.
-tickrate 128 Define o servidor local para 128 ticks, mas só é válido em servidores de treino.
-threads x Torna possível o uso dos threads do processador; substitua x pelo número desejado.
-fps_max_menu 999 Limita o FPS máximo do menu do jogo. Útil para evitar grandes oscilações de FPS.
-freq X Define a taxa de atualização do jogo (recomenda-se utilizar a mesma do monitor).
-nojoy Impede o uso de controles no CS, potencializando o espaço RAM para o CS:GO.
-mat_queue_mode 2 Força o computador a utilizar a renderização multinuclear.
-fps_max 999 Limita o FPS máximo que o jogo pode rodar. Útil para evitar grandes oscilações de FPS.
-forcenovsync Desabilita o VSync.
-console Habilita o uso do console através do '.
-w X -h X Muda a resolução do jogo; exemplo: -w 1920 e -h 108 coloca o jogo com 1920x1080.
